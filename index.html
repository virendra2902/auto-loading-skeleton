AutoSkeleton Props
Prop Type Default Description
loadingrequired boolean — When true, renders the skeleton. When false, renders children as-is.
animation 'shimmer' | 'pulse' | 'wave' | 'none' 'shimmer' Animation style applied to all skeleton blocks.
count number 1 Repeats the skeleton pattern N times. Perfect for list placeholders.
theme ThemeObject {} CSS variable overrides: baseColor, shimmerColor, duration, borderRadius.
className string '' Extra CSS class added to the wrapper element.
children ReactNode — The real component(s) to render when loading is false.
Additional Exports
🦴
SkeletonBlock
Single configurable block. Props: width, height, shape (circle/pill/rounded), animation.
📝
SkeletonText
Multi-line text placeholder. Control lines, lastLineWidth, lineHeight, gap.
👤
SkeletonAvatar
Circular avatar placeholder. Pass a size prop for quick setup.

Hook returning {loading, setLoading, skeletonProps} for ergonomic usage.
🪄
withSkeleton

Basic Usage
ProductPage.jsx
import { AutoSkeleton } from 'auto-loading-skeleton';

function ProductPage() {
  const { loading, data } = useFetch('/api/product');

  return (
    <AutoSkeleton loading={loading} animation="shimmer">
      <ProductCard product={data} />
    </AutoSkeleton>
  );
}

UserProfile.jsx
import { AutoSkeleton, useSkeleton } from 'auto-loading-skeleton';

function UserProfile() {
  const { loading, setLoading, skeletonProps } = useSkeleton(true);

  useEffect(() => {
    fetchUser().then(() => setLoading(false));
  }, []);

  return (
    <AutoSkeleton {...skeletonProps}>

withSkeleton HOC
EnhancedCard.js
import { withSkeleton } from 'auto-loading-skeleton';

const EnhancedProductCard = withSkeleton(ProductCard, {
  animation: 'shimmer',
  count: 1
});

// Usage: pass loading as prop directly
<EnhancedProductCard loading={loading} product={data} />

Node Type Detection
Source element Skeleton shape Detection signal
String / number child gray lines typeof children === 'string'
<h1>–<h6> heading wider line tag matches /^h[1-6]$/
<img> rect tag === 'img'
Avatar img / .avatar class circle className.includes('avatar')
<button> / role=button pill tag === 'button' || role
<input> / <textarea> rounded tag === 'input/textarea'
<svg> / .icon class small circle tag === 'svg' || .icon
Container <div> layout wrapper — recurse Traverses children
